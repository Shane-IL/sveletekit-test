About

This is a small personal project for me to get acquinted with SvelteKit. It will use some open APIs to display data about space rockets. I recently made a large webapp for work using NextJS and at the same time started to get into Svelte and quite liked the concept of SvelteKit so here we go.

Technologies used:

  • SvelteKit (obviously)
  • Tailwind CSS

Data sourced from:

  • r/Spacex open API
